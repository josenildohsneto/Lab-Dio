# Lab-Dio
Laboratorio de desafios da DIO

## Desafio - Trabalhando com Machine Learning na Prática no Azure ML
Neste desafio pude colocar em prática o aprendizado do primeiro módulo da Formação Microsoft Azure AI Fundamentals (AI-900).

### Passo a passo
1. Analisei o modelo exibido em aula.
2. Acessei a documentação do [Microsoft Learning ](https://microsoftlearning.github.io/mslearn-ai-fundamentals/Instructions/Exercises/01-machine-learning.html)
3. Segui o passo a passo da documentação da Microsoft para configurar o ML Automatizado.
4. Após a configuração, iniciei a execução do Modelo que executou por 26 min.
5. Após a execução iniciei a analise de modelo que também está disponível na documentação da microsoft.
6. Por último, gerei os pontos de extremidade a partir da implantação do melhor modelo.
